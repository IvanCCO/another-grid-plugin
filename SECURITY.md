# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| Latest release | Yes |
| Older releases | No |

## Reporting a vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately via GitHub's [Security Advisories](../../security/advisories/new) feature.

Include:
- A description of the vulnerability and its potential impact
- Steps to reproduce
- Any suggested fix (optional)

You will receive a response within 48 hours. If confirmed, a patch will be released as soon as possible and you will be credited in the release notes (unless you prefer otherwise).

## Scope

This extension runs on regular `http` and `https` pages. It does not collect, transmit, or store any user data beyond the grid preferences saved locally through Chrome sync storage. Security issues related to unintended data collection or privilege escalation are in scope.
